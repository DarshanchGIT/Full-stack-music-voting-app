Pausing a Container: This will stop the container's processes but keep the container running in the background.


docker pause <container_id>
Resuming a Paused Container:


docker unpause <container_id>

Stopping a Container: This will stop the container entirely. The next time you want to use it, you'll need to start it again.


=>docker stop <container_id>


Starting a Stopped Container:
=> docker start <container_id>
